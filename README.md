# kali-recon-toolkit
my beginner cybersecurity toolkit built with kali Linux

This project documents my hands-on practice with basic reconnaissance tools in Kali Linux as I begin my cybersecurity journey.

## Tools Covered
- Nmap
- Whois
- Nslookup
- theHarvester
- WhatWeb

## Goal
To build practical cybersecurity skills, document my learning, and grow my portfolio through consistent hands-on practice.
